# vue-js-pwa-sample

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run with docker
```
docker compose up
docker compose down --remove-orphans

docker-compose -f docker-compose-prod.yml up
```
